## How to setup connection

<Steps>
  <Step title="Login to your Trello account" />

  <Step title="Open a new tab and go to ttps://trello.com/power-ups/admin/">
    ![](/images/integrations/trello001.png)
  </Step>

  <Step title="Click &#x22;New&#x22;">
    ![](/images/integrations/trello002.png)
  </Step>

  <Step title="Fill up the fields (Iframe connector URL is optional)">
    ![](/images/integrations/trello003.png)
  </Step>

  <Step title="Click &#x22;Create&#x22;">
    ![](/images/integrations/trello004.png)
  </Step>

  <Step title="Click &#x22;Generate a new API key&#x22;">
    ![](/images/integrations/trello005.png)
  </Step>

  <Step title="Click &#x22;Generate API key&#x22;">
    ![](/images/integrations/trello006.png)
  </Step>

  <Step title="On the &#x22;Allowed origins&#x22;, add &#x22;https://app.hierarch.io&#x22; (w/o trailing slash)">
    Then click "Add"
    Copy the API key

    ![](/images/integrations/trello007.png)
  </Step>

  <Step title="Go to your integrations panel and paste the API key">
    Click "Get Token"

    ![](/images/integrations/trello008.png)
  </Step>

  <Step title="A pop-up will open, scrolldown and click Allow">
    ![](/images/integrations/trello010.png)
  </Step>

  <Step title="Click &#x22;Connected to App&#x22; checkbox">
    Then hit Save

    ![](/images/integrations/trello011.png)
  </Step>

  <Step title="Go to one of your boards">
    On the URL, add .json at the very end
    Copy the workspace id
    (Tip: You might need to install this [chrome extension](https://github.com/tulios/json-viewer/tree/0.18.0))

    ![](/images/integrations/trello012.png)
  </Step>

  <Step title="Copy the board id">
    ![](/images/integrations/trello013.png)
  </Step>

  <Step title="Paste these on the Settings tab, on the integrations panel">
    ![](/images/integrations/trello014.png)
  </Step>

  <Step title="Go to Actions tab">
    And click the checkboxes you want
    Then hit Save

    ![](/images/integrations/trello015.png)
  </Step>
</Steps>
